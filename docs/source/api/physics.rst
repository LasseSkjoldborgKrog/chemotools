chemotools.physics
==================

.. currentmodule:: chemotools.physics

Physical unit conversion methods for spectral intensity data. These transformers convert between common measurement representations used in vibrational and diffuse reflectance spectroscopy.

**Import from this module:**

.. code-block:: python

   from chemotools.physics import (
       IntensityConversion,
   )
